# Delivery Process & Coverage Images Guide

## Image Placeholders

### 1. "How It Works" Section Images

The "How It Works" section includes three image placeholders that you can replace with your own images or AI-generated images.

### Required Images:

1. **Step 1 - Book Delivery**
   - **Location**: `assets/img/gallery/delivery-step1.jpg`
   - **Recommended Size**: 400px width × 200px height
   - **Content**: Image showing someone booking/creating a delivery order (e.g., person on phone/computer, package being prepared)

2. **Step 2 - Track Package**
   - **Location**: `assets/img/gallery/delivery-step2.jpg`
   - **Recommended Size**: 400px width × 200px height
   - **Content**: Image showing package tracking (e.g., rider with package, tracking app on phone, package in transit)

3. **Step 3 - Fast Delivery**
   - **Location**: `assets/img/gallery/delivery-step3.jpg`
   - **Recommended Size**: 400px width × 200px height
   - **Content**: Image showing successful delivery (e.g., package being delivered, happy customer receiving package, delivery person at door)

### Image Requirements:

- **Format**: JPG or PNG
- **Aspect Ratio**: 2:1 (width:height)
- **File Size**: Optimized for web (under 200KB each)
- **Style**: Modern, professional, matching your brand colors

### AI Image Generation Suggestions:

You can use AI image generators like:
- DALL-E
- Midjourney
- Stable Diffusion
- Canva AI

**Prompt Examples:**
- Step 1: "Professional delivery service, person booking package delivery online, modern office, Nairobi Kenya, vibrant colors"
- Step 2: "Package tracking, delivery rider with smartphone showing tracking app, urban Nairobi setting, professional photography"
- Step 3: "Successful package delivery, happy customer receiving package, delivery person, modern residential area, bright and positive"

### Fallback:

If images are not found, icon placeholders will automatically display. The section will still look great!

- **Style**: Modern, colorful map visualization with highlighted coverage areas

- "Nairobi delivery service coverage map, showing Westlands, Karen, Eastlands, Kasarani areas highlighted, modern cartographic design, orange and blue color scheme"
