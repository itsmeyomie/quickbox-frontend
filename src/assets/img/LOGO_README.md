# Logo Placeholder Information

## Logo Dimensions

### Header Logo
- **Location**: `assets/img/logo/logo.png`
- **Exact Dimensions**: 180px width × 60px height
- **Format**: PNG (recommended) or SVG
- **Usage**: Displayed in the main navigation header

### Footer Logo
- **Location**: `assets/img/logo/logo2_footer.png`
- **Exact Dimensions**: 150px width × 50px height
- **Format**: PNG (recommended) or SVG
- **Usage**: Displayed in the footer section

## How to Add Your Logo

1. **Create your logo files** with the exact dimensions specified above
2. **Save them** as:
   - `logo.png` (for header) - 180px × 60px
   - `logo2_footer.png` (for footer) - 150px × 50px
3. **Place them** in the `src/assets/img/logo/` directory
4. **Replace the existing files** with your logo files

## Image Requirements

- **Format**: PNG (with transparency) or SVG (recommended for scalability)
- **Background**: Transparent background recommended
- **Color Mode**: RGB
- **Resolution**: 72-150 DPI (web standard)

## Notes

- The logo will automatically scale to fit the container while maintaining aspect ratio
- If the logo files are not found, a fallback text logo will be displayed
- For best results, use SVG format for crisp display on all screen sizes
